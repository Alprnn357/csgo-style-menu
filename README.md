# CSGO Style Menu For CS16Client Android
Features:
- CSGO-like design
- Easy to use
- Scoreboard in Team menu
- All weapons with prices
- Primary and Secondary ammo in one button

# Installation
Put the downloaded .pak file to your cstrike folder (game data).
